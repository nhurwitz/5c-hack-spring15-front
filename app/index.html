About Wikipath

Wikipath finds the shortest path from one Wikipedia page to another. That is, it shows you the minimum number of pages you need to click on to get from one (the "Starticle") to another (the "Endicle"). This can be useful for high scores in the Wikipedia Game or just for exploring Wikipedia.

It is built and maintained by Pomona students Noam Hurwitz, Dima Smirnov, Ziv Epstein and Argo developer Ted Kornish and was originally implemented for the Claremont 5C Hackthon where it won first prize. We did it by repersenting Wikipedia as a directed graph and running a modified Djiskra's algorithm on it. We hope you like it!
